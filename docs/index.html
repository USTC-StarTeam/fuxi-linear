1. Method At A Glance

FuXi-Linear separates semantic, temporal, and positional signals for long-term time-aware sequential recommendation while keeping linear complexity for thousand-length user histories.

1. Temporal

Temporal Retention Channel captures period-aware timestamp signals.

2. Linear

Linear Positional Channel restores positional information without quadratic attention.

3. Gin

Gin configs cover KuaiRand-27K, KuaiRec, and MovieLens-20M experiments.

2. Code And Usage

Start from the repository commands below, then follow README details for data paths, model checkpoints, and evaluation outputs.

pip3 install gin-config absl-py scikit-learn scipy matplotlib numpy
python3 preprocess_public_data.py
CUDA_VISIBLE_DEVICES=0,1 python3 main.py --gin_config_file=configs/kuairand-27k/linear-4b-l1024-b64x2.gin
